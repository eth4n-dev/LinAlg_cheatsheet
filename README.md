# Linear Algebra Cheatsheet

A comprehensive cheatsheet for Linear Algebra covering key concepts and topics.

## Sections

### Part I: Foundations
- **Vectors** - Vector operations, linear combinations, scalar products, norms, and angles
  - Vector addition and scalar multiplication
  - Scalar products and Euclidean norms
  - Cauchy-Schwarz inequality and triangle inequality
  - Linear independence and span

### Part II: Advanced Topics
- Matrices and matrix operations
- Vector spaces and subspaces
- Bases and dimensions
- Eigenvalues and eigenvectors
- Diagonalization
- Orthogonality and projections
- And more...

## Files

- `main2.tex` - Main LaTeX cheatsheet document
- `notes_part_I.txt` - Detailed notes for Part I (vectors and foundations)
- `notes_part_II.txt` - Detailed notes for Part II (advanced topics)

## Usage

Compile the LaTeX document with:
```bash
pdflatex main2.tex
```
